# 读懂一个项目

<h2 id="goal">任务目标</h2>

让你已经安装的 AI 工作台先看懂项目，不改代码。

<h2 id="prepare">准备物品</h2>

- 已安装 [Codex](/docs/runtime/codex) 或 [Claude Code](/docs/runtime/claude-code)
- 一个你想理解的项目文件夹

<h2 id="steps">通关步骤</h2>

1. 打开你已经安装的 AI 工作台，比如 `Codex App`、`Claude Code` 或 VS Code 插件
2. 选择或进入你想理解的项目文件夹
3. 新建一次对话
4. 复制下面这句话


<h2 id="prompt">复制这句话</h2>

```text
先不要改代码。请阅读这个项目的目录结构，告诉我它是做什么的、入口文件在哪里、我应该先读哪些文件。
```

<h2 id="done">通关标志</h2>

它能说清：项目用途、入口文件、核心目录和推荐阅读顺序。

<h2 id="stuck">卡关怎么办</h2>

如果它上来就想改文件，提醒它：`先只读项目，不要修改任何文件。`
