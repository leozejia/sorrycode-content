# Understand a Project

<h2 id="goal">Goal</h2>

Ask your installed AI workbench to understand the project before changing code.

<h2 id="prepare">What You Need</h2>

- [Codex](/docs/runtime/codex) or [Claude Code](/docs/runtime/claude-code) installed
- a project folder you want to understand

<h2 id="steps">Steps</h2>

1. Open your installed AI workbench, such as `Codex App`, `Claude Code`, or a VS Code extension
2. Choose or enter the project folder you want to understand
3. Start a new thread
4. Copy the prompt below


<h2 id="prompt">Copy This</h2>

```text
Do not change code yet. Read this project structure and tell me what it does, where the entry points are, and which files I should read first.
```

<h2 id="done">Completion Check</h2>

It can explain the project purpose, entry files, key folders, and suggested reading order.
