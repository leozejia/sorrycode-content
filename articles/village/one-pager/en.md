# Understand One-pagers

<h2 id="goal">Goal</h2>

Understand what a `one-pager` is and when to use it.

<h2 id="what-is-it">What It Is</h2>

A `one-pager` is a finished one-page artifact that explains one thing clearly.

It can be:

- a product intro
- a personal intro
- a project brief
- an event proposal
- a short page for sharing, pitching, or reporting

<h2 id="prepare">What You Need</h2>

- [Kami](/docs/skills/kami) installed
- raw text you want to turn into one page

<h2 id="prompt">Copy This</h2>

```text
First help me decide whether the content below is suitable for a one-pager. If yes, tell me which sections the page should include. Do not generate the final version yet.
```

<h2 id="done">Completion Check</h2>

You can explain the one-sentence goal, audience, and sections of the page.
