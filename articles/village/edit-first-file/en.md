# Edit Your First File

<h2 id="goal">Goal</h2>

Make one small safe edit while keeping the habit of planning first.

<h2 id="prepare">What You Need</h2>

- You have completed `Understand a Project`
- You know one small thing you want to change

<h2 id="prompt">Copy This</h2>

```text
I want to make a very small change. Do not edit files yet. First tell me which files you will read, how you plan to change them, and what risks there may be. Wait for my confirmation before editing.
```

<h2 id="done">Completion Check</h2>

It gives a plan first and only edits the small area you asked about.
