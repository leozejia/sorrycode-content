# 修改第一个文件

<h2 id="goal">任务目标</h2>

完成第一次安全的小改动，建立“先计划，再动手”的习惯。

<h2 id="prepare">准备物品</h2>

- 已经完成 `读懂一个项目`
- 知道你想改哪个小地方

<h2 id="steps">通关步骤</h2>

1. 先让 AI 工作台解释它准备怎么改
2. 确认计划没问题
3. 再允许它修改文件
4. 改完让它总结改了哪里

<h2 id="prompt">复制这句话</h2>

```text
我想改一个很小的地方。请先不要动文件，先告诉我你准备读哪些文件、怎么改、可能有什么风险。等我确认后你再修改。
```

<h2 id="done">通关标志</h2>

它先给出计划，并且只修改了和任务相关的小范围文件。

<h2 id="stuck">卡关怎么办</h2>

如果改动范围变大，让它停下：`请收窄范围，只处理我刚才说的这个小问题。`
