# Huashu Design

`Huashu Design` is a third-party design skill.

Use it for more complex design deliverables: app prototypes, HTML slides, editable PPTX, motion design, infographics, design variations, and expert critique.

If `Kami` is for polished documents and paper-like artifacts, and `Magazine Web PPT` is for web-style presentations, `Huashu Design` brings the agent into a fuller designer workflow: ask questions, collect brand assets, propose directions, produce, and critique.

<h2 id="what-it-does">What It Does</h2>

Common uses:

- clickable app and web prototypes
- HTML decks and editable PPTX
- product launch motion, MP4, and GIF
- infographics and data visualization
- side-by-side design directions
- five-dimensional critique and revision advice

<h2 id="install">Install</h2>

Make sure you have installed `Codex` or `Claude Code` first.

### Codex

```bash
npx skills add alchaincyf/huashu-design -a codex -g -y
```

### Claude Code

```bash
npx skills add alchaincyf/huashu-design -a claude-code -g -y
```

If the installer or upstream command changes, follow the official repository: <https://github.com/alchaincyf/huashu-design>

<h2 id="first-prompt">First Prompt</h2>

```text
Use Huashu Design to make a clickable 4-screen iOS prototype for a Pomodoro app. Ask the necessary questions first, then give me 3 design directions to choose from.
```

For motion:

```text
Use Huashu Design to turn this product launch logic into a 30-second motion piece. Give me the script and visual direction first; do not generate the final file yet.
```

For critique:

```text
Use Huashu Design to run a five-dimensional expert critique on this page. Tell me what to keep, what to fix, and the quickest improvements.
```
